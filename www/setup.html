Installation

npm install md-out-click

Using the old way

Include the script in between angular source script & your app specific scripts-

Using with Webpack

Import and include mdOutClick module in your webpack entry file-

// entry.js import mdOutClick from 'mdOutClick'; // ES6 require('mdOutClick'); // ES5

Finally define mdOutClick as dependency in your angular app declaration-

angular.module('yourApp', [ // other dependencies 'mdOutClick' ]);
