Using with Side Navigation

Problem:

You are showing a side navigation panel, and want to close it when clicked anywhere outside.

Solution:
// Include mdOutClick in app dependency angular.module('yourApp', [ // other dependencies 'mdOutClick' ]); // In your template
  • Side Nav Menu 0
  • Side Nav Menu 1
// Inside controller $scope.isMenuClose = true; $scope.close = function () { $scope.isMenuClose = true; }; $scope.open = function () { $scope.isMenuClose = false; };

Or using the OutClickService by defining sidenav as your own directive following a similar approach as explained in dropdown example.
