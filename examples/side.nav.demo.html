Using with Side Navigation

Problem:

You are showing a side navigation panel, and want to close it when clicked anywhere outside.

Solution: 

      
        // Include mdOutClick in app dependency
        angular.module('yourApp', [
          // other dependencies
          'mdOutClick'
        ]);

        // In your template
        <ul class="nav flex-column" on-out-click="close()">
          <li class="nav-item">
            <a href="#" class="nav-link">Side Nav Menu 0</a>
          </li>

          <li class="nav-item">
            <a href="#" class="nav-link">Side Nav Menu 1</a>
          </li>
        </ul>


        // Inside controller
        $scope.isMenuClose = true;

        $scope.close = function () {
          $scope.isMenuClose = true;
        };

        $scope.open = function () {
          $scope.isMenuClose = false;
        };
Or using the OutClickService by defining sidenav as your own directive following a similar approach as explained in dropdown example.
